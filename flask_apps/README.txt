This folder contains small flask apps

=====================================

app.py
this runs a todo app that allows users
to add and remove text items from a list
this was built using the following
tutorial:
https://www.digitalocean.com/community/tutorials/how-to-use-mongodb-in-a-flask-application

kanji.py
this runs a dictionary app that allows
users to create tags for various kanji

=====================================

to switch apps run this CMD
export FLASK_APP=kanji
or
export FLASK_APP=app

then run this CMD
flask run
